Yam tv


An android movies and tv-shows app that scrapes the web for video links for free.The app uses the TMDB api along with IMDB to get movies and episodes information then it scrapes the web using the MVS api along with some other apis to get the best quality and fast server links


The app is 100% auto as it uses the AI technology to show the user movies based on their latest watches. it also provides an easy-to-use API that can be found here to get the movies to link directly in the web

SuperPlayer


SuperPlayer is mainly a web video player that can play videos hosted in bunch of video hosting services like Google Drive,Youtube,Ok.ru, Upstream... and much much more.Superplayer has also the ability to give the download link and it uses the OpenSubtitles API to provide subtitles with any language to the user.


Superplayer was an android app at first but due to the request of many users it became a web app that offers custom players for the premium users and the android app name was converted to Bytestream
