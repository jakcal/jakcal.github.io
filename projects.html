SimpleQuran

Mobile-first PWA for listening to the Quran fully offline. Pick a reciter, tap a surah, and it caches quietly in the background so it plays with zero connection. Installs to the home screen like a native app—no app store needed. The fun part was the media layer: service worker caching, background downloads with per-surah and bulk 'download all 114' progress, and an install/offline flow that holds up on a flaky mobile connection. Calm dark UI, bilingual (AR/EN) surah list with verse counts, search by name or number, and an Offline filter.

React
